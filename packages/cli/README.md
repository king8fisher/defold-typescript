# defold-typescript

The end-user CLI for scaffolding and building [Defold](https://defold.com/)
projects written in TypeScript, transpiled to Lua.

```sh
bunx defold-typescript init    # add TypeScript to a Defold project, or scaffold a new one
bunx defold-typescript build   # transpile src/ to Lua alongside your Defold sources
bunx defold-typescript watch   # rebuild on change
```

Installs the binary `defold-typescript`:

```sh
npm i -D @defold-typescript/cli
```

See the repository [README](https://github.com/king8fisher/defold-typescript#readme)
and [`docs/guide/`](https://github.com/king8fisher/defold-typescript/tree/main/docs/guide)
for the full workflow.
